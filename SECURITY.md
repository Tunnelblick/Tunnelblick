# Security Policy

## Reporting a Vulnerability

 * [Report a security vulnerability using GitHub](https://github.com/tunnelblick/tunnelblick/security/advisories/new).

and/or

 * [Report a security vulnerability via email](mailto:developers@tunnelblick.net). You can expect a response within 7 days.


## Supported Versions

| Version     | Supported          |
| -------     | ------------------ |
| 3.8.8d      | :white_check_mark: |
| 4.0.0beta08 | :white_check_mark: |
| < 3.8.8d    | :x:                |
